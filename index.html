Summer 2016: Personal Projects with Python

Computer Science Program for High School Girls

Congratulations to our Summer 2016 Python Program Members!

Bluesoxx hosts computer science programs for high school girls in Vancouver, BC.  This summer, we are completing custom personal projects using the Python programming language.

Congratulations to Alice, Jessica, Kaylie, Lucia, and Madeleine (1st Squad), plus Gabriella, Karat, Lovely, and Sydney (2nd Squad) on finishing the Bluesoxx weeklong Python bootcamp, July 11-15!

Our members have already completed some fabulous projects, and we can all keep extending them further over the summer:

  • A program that not only generates Mad Libs stories by combining user inputs with pre-constructed templates, but also uses the NLTK natural language processing library to break down arbitrary texts from the internet into parts of speech, generating new templates automatically.  Next step: fill out the templates automatically, too, finding appropriate words via frequency analysis, and synthesize news stories or political speeches automatically.
  • A classifier that uses the OpenCV computer vision module to automatically detect people's faces from pictures.  Next step: recognize faces and tag pictures automatically by who is in them.
  • An image processor that takes a photo and converts it into ASCII art with custom resolution.  Next step: color images and animation.
  • A custom Chrome extension that pops up within any web page and allows the user to draw graphics using the Processing language for computer-generated art.  Next step: drawing over the web page that the user is currently viewing.
  • A world map infographic where countries are automatically colored through the user's grouping, according to United Nations subcommittees.  Next step: the program groups the countries and shades their colorings on its own according to world demographic data.
  • A web site for displaying all of our class projects, using responsive CSS code to ensure that all formatting is resized automatically across devices with different screen sizes.  Next step: running project code as containers within the browser, to allow live demos over the web.
  • One program that plays a game where the user must discover a secret by a series of questions, plus a second program that in turn guesses the user's secret.  Next step: make the games play each other a thousand times and study the results.
  • Cryptographic package for encyphering and decyphering text messages using the Caesar cypher (easy to break by brute force) and the Vigenère cypher (keyspace too large to break by brute force).  Next step: breaking the Vigenère cypher using frequency analysis.

About Bluesoxx

Past Cohorts

For May and June, Bluesoxx worked through introductory lessons in graphics, programming languages, signal processing, networks, and security.   Congratulations to Anastasia, Christel, Emily, Jessica, Julia, Lovely, Madeleine, Nicole, Rachel, Sara, Sydney, and Tina, for completing Stanford CS101!


First Class Ambitions

We will definitely make some fun and simple demo projects.   But overall, the class is about real fundamentals in formal computer science, and professional-level programming.   We will also work on leadership, presentation skills, and health and fitness: less crafting-for-kids and more being-a-boss.

Getting it Done Together

The Bluesoxx program combines online materials from university-level computer science courses with personalized instruction by a local expert.   Most importantly, though, students work together to build and share expertise as a team, and take the lead in making this happen: not quite like joining a sorority, but more than just taking a class.

Real Accomplishments

The ability to use software, like the ability to read 300 years ago, is transforming assumptions about what people can accomplish.   Early birds who progress from computer-literate to best-selling-authors of computation are not just guaranteeing the income or lifestyle they want--they have the opportunity to shape the world.
